O que eu sei...

HTML

Usado para construção de sites, é o "esqueleto" do site.

CSS

Usado para estilização de sites, é toda parte visual presente no site.

JavaScript

Usado para dar funcionalidade ao site, é o "cérebro" do site.

JQuery

Uma biblioteca para JavaScript, é utilizaada para simplicar o código.

NodeJS

Uma biblioteca para JavaScript, é utilizada para fazer uma interação com o banco de dados.

PHP

Usada no lado do servidor, é utilizada para fazer uma interação com o banco de dados.

Python

Uma linguagem "universal", é utilizada para sistemas para web e para desktop.

MySQL

Uma linguagem de banco de dados, é utilizada para criar onde os dados serão inseridos e gerenciar esses dados.
